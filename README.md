# 🏥 Hospital Management System (Java + Swing + MySQL)

This is a desktop-based Hospital Management System built using Java (Swing for GUI), MySQL for database management, and JDBC for connectivity. It simulates basic hospital operations like room allocation, ambulance tracking, patient details management, and login authentication.

> 🚀 The project includes a clean graphical user interface and supports interaction with a MySQL backend for real-time data updates.

---

## 🔧 Technologies Used

- **Java SE 8+**
- **Java Swing** for building GUI
- **MySQL** as the relational database
- **JDBC** for Java-MySQL connectivity
- **rs2xml.jar** for converting `ResultSet` to `JTable`
- **IDE**: IntelliJ IDEA / VS Code

## ✅ Features

- Admin Login system with authentication
- Add/View Patient Information
- Check Room Availability (Available / Occupied)
- Track Ambulance Availability and Location
- GUI Interface built with Java Swing
- Data persistence using MySQL
- Modular class-based structure
- Proper use of MVC components (Model: MySQL, View: Swing, Controller: JDBC)
